Constitutional Agent SDK

An open-source (MIT/Apache 2.0) infrastructure toolkit that wraps any standard LLM inside a thermodynamic Viability Kernel. Submitted to ARIA Scaling Trust Track 2 (£600k, March 2026).

constitutional-compiler
Translates natural language security policies into deterministic, machine-executable graph constraints via Frame-Based Principled Reasoning (FBPR). Binds Phenotype (human intent) to Genotype (TypeScript hard constraints). Policy violation becomes structurally impossible rather than statistically unlikely.
rho-monitor
Tracks causal saturation ρ = L/C in real time. Detects the η-pump — adversarial load injection without capacity increase — and triggers homeostatic repair before constitutional violations occur. Tiered computation: expensive evaluation triggered only when epistemic integrity is under threat.
constitutional-halt
Safe transfer of authority when the Viability Kernel cannot be restored. Refuses action outside constitutional mandate, executes Terminal Gait, transfers control to human operator, emits complete deterministic audit log. The architectural opposite of Silent Failure.
